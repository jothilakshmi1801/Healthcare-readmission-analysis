# Healthcare-readmission-analysis
Healthcare Readmission & Resource Optimization Dashboard

## Project Overview
This project analyzes patient readmission patterns and helps optimize healthcare resources using SQL and Power BI.

## Tools Used
- SQL (Oracle)
- Excel
- Power BI

## Key Insights
- Readmission by age group
- Readmission by region
- Discharge analysis
- Patient risk insights

## Dashboard Pages
- This project contains a single interactive dashboard page with multiple visualizations and filters for analysis.
